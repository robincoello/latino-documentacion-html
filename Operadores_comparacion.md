### Igualdad
```
escribir(l == falso)  #falso
escribir(e == 10)     #verdadero
escribir(d == 3.0)    #falso

```
### Desigualdad
```
escribir(l != falso)  #verdadero
escribir(e != 10)   #falso
escribir(d != 3.0)   #verdadero
escribir(lit != 'hola')	  #verdadero
escribir(cad != "esto es una 'cadena'")  #falso
```
