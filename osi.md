# Osi

Este se usa cuando se tiene varias opciones y se desea comprar con una de ellas, al final se pone un `sino`, en el caso que no haya ninguna opcion equivalente

```
// dia
d = 2

si ( d == 1)
    escribir("Es lunes")
osi d == 2 
    escribir("Es martes")
osi d == 3
    escribir("Es miercoles")
osi d == 4
    escribir("Es jueves")
osi d == 5
    escribir("Es viernes")
osi d == 6
    escribir("Es sábado")
osi d == 7
   escribir("Es domingo")
sino 
   escribir("Día desconocido")
fin
```
